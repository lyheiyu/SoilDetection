Field dictionary — Practice Catalog

Practice_ID (text)
P001–P038. Locked list for catalog consistency.
Name (text)
Human-readable label, e.g., “Minimum/No tillage”.
Category (enum)
Soil management / Nutrient / Water / Crop diversification / Organic amendments / Chemical applications / Remediation / Compaction / Erosion control / Other.
Description (text)
Short definition of the practice.
Expected effects (text)
Outcomes on soil/yield/environment.
Typical operations (text)
Canonical actions, tools, or steps.
Recommended indicators (list)
Indicator IDs from core list.
Cost (number)
Rough €/ha if known.
Notes (text)
Links / references / constraints.

Example: P006 “Cover crops (C sequestration)” — Typical operations: sow rye at 150 kg/ha post-harvest; terminate by roller in spring.

Field dictionary — TreatmentPractices / PracticeOperations

Experiment ID (text)
Link to Trials/Treatments.
Treatment number (integer)
0=Control, 1=T1, 2=T2, 3=T1+T2, etc.
Practice_ID (enum)
Reference to Catalog P001–P038.
Operation name (text)
Granular action, e.g., “Apply compost”.
Date / Timepoint
Calendar date and/or project phase (T0/T1/T2/Tf).
Rate & Unit
Numeric quantity and unit.
Method (text)
How it was applied (tool/depth etc.).
Notes (text)
Any additional context.

Example: NL-02 · T1 · P006 · “Drill winter rye” · Date 2025-10-05 · Rate 150 · Unit kg/ha · Method no-till drill 3 cm.

At least one indicator required here or in extra IDs.
